## Overview

This portfolio website is built with Jekyll (al-folio fork) and deployed at [https://richardhcli.com](https://richardhcli.com). It features a clean, responsive design with a timeline-based project showcase.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Timeline View**: Projects are displayed in a chronological timeline
- **Automatic Deployment**: GitHub Actions handles build and deployment
- **Technical Blog**: Engineering write-ups alongside project pages

## Technologies Used

- Jekyll 4.x
- Liquid templates & SCSS
- GitHub Actions
- GitHub Pages / custom domain

## Getting Started

To run this site locally:

```bash
docker compose pull
docker compose up
```

Then visit `http://localhost:8080` in your browser.
